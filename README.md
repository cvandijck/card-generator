# card-generator

Generate fun artificial cards for holidays or other occassions

More information can be found on the project's [website](https://github.com/cvandijck/card-generator).

## Installation

*Add information on how to install card_generator.*

You can install card_generator using the distributed wheel via pip:
```console
python -m pip install card_generator-0.0.0-py3-non-any.whl
```

This installation should automatically pull and install all required dependencies from PyPI. In the case of custom 
dependencies, you can direct pip to a local server:

```console
python -m pip install card_generator-0.0.0-py3-none-any.whl --find-links .\dependencies
```

## Getting Started

*Add information on the typical use of card_generator, either as library or as a console script.*
